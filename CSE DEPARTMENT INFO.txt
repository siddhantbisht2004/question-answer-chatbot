// course_catalog.txt
Computer Science and Engineering Department - Course Catalog 2024

Core Courses:
1. CS101: Introduction to Computer Programming
Duration: 1 semester
Credits: 4
Description: Fundamental concepts of programming using Python. Topics include variables, data types, control structures, functions, and basic object-oriented programming.
Prerequisites: None
Learning Outcomes:
- Write basic Python programs
- Understand programming fundamentals
- Implement simple algorithms

2. CS201: Data Structures and Algorithms
Duration: 1 semester
Credits: 4
Description: Implementation and analysis of fundamental data structures and algorithms. Topics include arrays, linked lists, trees, graphs, sorting, and searching algorithms.
Prerequisites: CS101
Learning Outcomes:
- Implement basic data structures
- Analyze algorithm complexity
- Design efficient solutions

3. CS301: Database Management Systems
Duration: 1 semester
Credits: 4
Description: Fundamentals of database design, development, and management. Covers SQL, normalization, transaction management, and database security.
Prerequisites: CS201
Learning Outcomes:
- Design normalized databases
- Write complex SQL queries
- Implement database applications

// lab_manual.txt
Computer Programming Lab - Manual and Guidelines

Lab Rules and Regulations:
1. Students must arrive on time for lab sessions
2. Maintain silence in the laboratory
3. No food or drinks allowed
4. Save your work regularly
5. Report any hardware/software issues to lab assistants

Python Programming Lab Exercises:

Exercise 1: Basic Python Programs
1. Write a program to find factorial of a number
2. Implement a calculator with basic operations
3. Create a program for student grade calculation

Exercise 2: Data Structures Implementation
1. Implement a linked list with basic operations
2. Create a binary search tree
3. Develop a graph representation

Exercise 3: Database Connectivity
1. Create a student database using SQLite
2. Implement CRUD operations
3. Develop a simple query interface
